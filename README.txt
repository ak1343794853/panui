在不使用ajax的情况下：

1. 上传文件功能

点击上传文件后弹出模态对话框，点击上传开始上传文件，这一部分请利用jquery文件上传插件实现

简单的说，就是把文件上传功能绑定到“上传”按钮上

注：此部分UI效果用bootstarp的插件实现，请参加文档：http://v3.bootcss.com/javascript/#modals


2. 新建文件夹功能

点击新建文件夹，弹出对话框，输入文件名，点击确定，刷新页面


3. 共享文件夹

实现同1、2，具体仍参考 http://v3.bootcss.com/javascript/#modals


4. 显示已删除文件

点击后刷新页面

--------------------------------

单个文件操作：

1. 下载 （事件绑定已经写在main.js中）

一种方案：给下载按钮绑定事件，点击时获取目前选中item中的下载链接，访问下载链接


2. 删除

参考上述模态对话框实现，删除时弹出对话框，确认后删除并刷新页面


3. 重命名

参考 新建文件夹功能，点击‘重命名’弹出对话框，输入新名字后，确定，刷新页面


4. 移动到和复制到

模态框的 modal-body 里面
